## 🚀 Deploy to GitHub Pages — Step by Step

### STEP 1 — Set Up VS Code
- Install [VS Code](https://code.visualstudio.com/)
- Install [Git](https://git-scm.com/downloads)
- Open VS Code → Install extension: **Live Server** (by Ritwick Dey)

### STEP 2 — Open Your Project
1. Put all 8 files in a folder called `sirius-handloom`
2. VS Code → **File → Open Folder** → select `sirius-handloom`
3. Right-click `index.html` → **Open with Live Server** to preview

### STEP 3 — Create GitHub Repo
1. Go to [github.com](https://github.com) → Login as `shajahanimdaad53`
2. Click **+** → **New Repository**
3. Name: `sirius-handloom`
4. Visibility: **Public**
5. ❌ Do NOT check "Add README" (you already have one)
6. Click **Create Repository**

### STEP 4 — Push Code via VS Code Terminal
Open Terminal in VS Code (`Ctrl + backtick`):

```bash
git init
git add .
git commit -m "🚀 Initial commit — Sirius Handloom website"
git branch -M main
git remote add origin https://github.com/shajahanimdaad53/sirius-handloom.git
git push -u origin main
```

### STEP 5 — Enable GitHub Pages
1. Go to: `https://github.com/shajahanimdaad53/sirius-handloom`
2. Click **Settings** (top tabs)
3. Left sidebar → **Pages**
4. **Source:** Deploy from a branch
5. **Branch:** `main` | **Folder:** `/ (root)`
6. Click **Save**

### STEP 6 — Your Live Website! 🎉
```
https://shajahanimdaad53.github.io/sirius-handloom/
```
*(Goes live in 1–3 minutes)*
